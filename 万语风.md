# 项目方案：「万语风」(PolyWind)

## 面向中国—东盟的多语言气象智能决策助手

---

## Context

团队选择大赛1赛道3"AI+众创气象服务"创新赛道·智能应用类。经过多轮调研和方向筛选，确定项目方向为**基于LLM Agent的多语言气象智能服务应用**，突出面向东盟的多语言交互能力。

- 报名截止：2026年7月31日
- 初赛材料提交：10月13日前
- 决赛：11月16-17日
- 团队技术栈：前端/Web开发 + AI/深度学习 + 数据/后端

---

## 项目定位

**「万语风」** 是一个基于大语言模型 + 气象数据融合的多语言智能气象服务Agent，将天气预报转化为面向公众和行业的可执行决策建议，覆盖广西及东盟地区，支持中文、英语、越南语、泰语、印尼语、缅甸语、老挝语等7种语言交互。

**Slogan**：万种语言，一种风 —— 以AI连接气象与世界

---

## 核心差异化

| 维度 | 现有产品（风和/阿福等） | 万语风 |
|---|---|---|
| 语言 | 仅中文 | 中/英/越/泰/印尼/缅/老 7语 |
| 交互方式 | 信息播报/被动查询 | 对话问答 + **AI主动推送** 双模式 |
| 服务深度 | 天气预报 | 预报→影响分析→行动建议 |
| 应用场景 | 单一场景 | 公众+行业+跨境多场景 |
| 技术架构 | 传统应用 | LLM Agent + 气象工具链 |
| 终端适配 | 仅Web | **Web + 移动端H5 + 微信小程序** |

## 六大功能模块

### 模块1：多语言智能天气问答
- 自然语言查询天气（"明天河内会下雨吗？"/"พรุ่งนี้กรุงเทพฝนจะตกไหม?"）
- 多语言语音输入/输出
- 基于RAG的气象知识科普（台风成因、预警信号含义等）
- 个性化天气播报生成（通勤/运动/旅行场景）
- **AI主动推送**：基于用户地理位置、出行习惯和关注行业，定时智能生成个性化气象提醒推送
  - 早间通勤推送："今天南宁有阵雨，建议带伞，越南河内晴好适合出行"
  - 行业定时推送："未来3天甘蔗产区高温预警，建议提前灌溉"
  - 灾害临灾推送："台风'XX'预计明日登陆越南北部，请提前做好防护"
  - 支持用户自定义推送频率（每日/每周/灾害触发）和关注领域

### 模块2：行业气象决策引擎
- **农业**：甘蔗/水稻/水果种植的灌溉、施药、收割时机建议
- **旅游**：景点天气舒适度评分、最佳出行时段推荐
- **物流**：跨境物流路径天气风险评估、运输时间窗口推荐
- **能源**：光伏发电窗口预测、风电出力建议

### 模块3：极端天气与自然灾害预警系统
- **多灾种覆盖**：台风/暴雨/洪涝/高温热浪/干旱/雷暴/大风/地质灾害等8类灾害
- **递进式预警**：灾前预判→临灾预警→灾中追踪→灾后评估 四阶段全链条
  - 灾前（提前72-24h）：基于数值模式+AI模型预判风险等级
  - 临灾（提前24-6h）：发布分级预警，推送防护建议
  - 灾中（实时）：滚动更新灾情动态，追踪路径和强度变化
  - 灾后（结束6h内）：影响评估报告，恢复建议
- **跨境灾害联动**：台风从南海生成→影响广西→登陆越南的全路径追踪，多语言预警同步推送
- **行业灾害影响评估**：
  - 农业："台风'XX'预计明日经过甘蔗产区，风力8级，建议提前收割成熟甘蔗"
  - 旅游："曼谷未来3天持续高温40°C+，建议调整户外行程"
  - 物流："北部湾海面大风7级，跨境航运受影响，建议推迟发货"
- **基于地理围栏的精准推送**：用户设定关注区域，灾害预警自动触达

### 模块4：气象数据可视化面板
- 基于地图的实时天气态势展示（广西+东盟）
- 行业气象指数看板（农业指数、旅游指数、物流风险指数等）
- 预警信息地图标注与传播链路可视化

### 模块5：AI定时推送引擎
- 基于用户画像的个性化推送
- 支持定时推送（早间/午间/晚间）
- 灾害触发式即时推送
- 行业专属推送（农业/旅游/物流）

### 模块6：跨境协同服务
- 多语言统一服务入口
- 跨境灾害信息共享
- 中国-东盟气象合作数据对接

## 多国气象数据API接入方案

### 一、全球通用API（覆盖东盟全部10国）

#### 1. Open-Meteo（首选，免费开源）
- **官网**：https://open-meteo.com
- **覆盖**：全球，含全部东盟国家
- **免费额度**：非商业用途每日10,000次调用，无需API Key
- **数据内容**：实时天气、逐小时预报（最长16天）、历史数据（80年）、降水/风/温度/湿度等全要素
- **数据源**：ECMWF（欧洲中期天气预报中心）+ GFS + 多国区域模型
- **接入方式**：纯HTTP GET请求，返回JSON，无需SDK
- **示例调用**：
  ```
  GET https://api.open-meteo.com/v1/forecast?latitude=21.03&longitude=105.85&hourly=temperature_2m,precipitation&timezone=Asia/Ho_Chi_Minh
  （越南河内天气预报）
  ```
- **优势**：免费、无需注册、覆盖全、数据质量高（基于ECMWF）
- **获取方式**：直接访问API即可，无需注册

#### 2. OpenWeatherMap（备选）
- **官网**：https://openweathermap.org/api
- **覆盖**：全球
- **免费额度**：每日1,000次调用，60次/分钟
- **数据内容**：当前天气、5天预报、空气质量、紫外线
- **接入方式**：REST API，需注册获取API Key
- **获取方式**：注册账号 → 获取API Key → 免费层自动激活
- **适合**：作为备用数据源和交叉验证

#### 3. Google Maps Weather API
- **官网**：https://developers.google.com/maps/documentation/weather
- **覆盖**：全球大部分国家（含泰国、柬埔寨、老挝等，越南覆盖有限）
- **数据内容**：当前天气、逐小时预报（240h）、每日预报（10天）、历史数据（24h）
- **接入方式**：REST API，需Google Cloud账号
- **获取方式**：创建Google Cloud项目 → 启用Weather API → 获取API Key
- **注意**：需付费，按调用量计费

### 二、中国气象数据（广西+华南区域）

#### 4. 中国气象数据网（权威数据源）
- **官网**：https://data.cma.cn
- **数据内容**：地面站观测、高空探测、卫星遥感、数值预报产品、再分析数据
- **开放目录**：12类106种气象数据产品（2025年最新发布）
- **接入方式**：Web下载 + API接口调用 + 图层服务
- **获取方式**：注册账号 → 实名认证 → 免费下载开放数据
- **特别价值**：中国气象局全球大气再分析产品、全球地面/高空数据集（首次国际共享）

#### 5. 和风天气API
- **官网**：https://dev.qweather.com
- **覆盖**：全球，中国数据最精细
- **免费额度**：每日1,000次调用
- **数据内容**：实时天气、7天预报、灾害预警、生活指数、空气质量
- **接入方式**：REST API，需注册获取Key
- **获取方式**：注册 → 创建应用 → 获取API Key
- **特色**：中文灾害预警信息、生活指数（穿衣/运动/紫外线）

### 三、东盟各国官方气象数据源

#### 6. 越南国家水文气象预报中心（NCHMF）
- **官网**：https://nchmf.gov.vn
- **数据**：越南区域天气预报、台风路径、降水预报
- **接入方式**：网页数据抓取 + 官方数据申请（可联系合作）

#### 7. 泰国气象局（TMD）
- **官网**：https://www.tmd.go.th
- **数据**：泰国天气预报、季风监测、极端天气预警
- **API**：提供部分开放数据接口
- **接入方式**：REST API + 网页数据

#### 8. 印尼气象气候地球物理局（BMKG）
- **官网**：https://www.bmkg.go.id
- **数据**：地震预警、天气预报、海啸预警、气候数据
- **API**：提供开放数据API（需申请）
- **特色**：地震+海啸预警数据（全球领先）

#### 9. 菲律宾大气地球物理和天文服务管理局（PAGASA）
- **官网**：https://www.pagasa.dost.gov.ph
- **数据**：台风预警（全球最详细的台风公报）、季风监测
- **接入方式**：网页数据 + RSS订阅

#### 10. 马来西亚气象局（MET Malaysia）
- **官网**：https://www.met.gov.my
- **数据**：天气预报、空气质量、极端天气预警
- **API**：提供部分开放接口

#### 11. 新加坡气象服务（MSS）
- **官网**：https://www.weather.gov.sg
- **数据**：高精度区域天气、闪电预警、空气质量
- **API**：提供开放数据API

### 四、极端天气与灾害预警专用数据源

#### 12. 台风/热带气旋数据
- **JTWC（美国联合台风警报中心）**：https://www.metoc.navy.mil/jtwc — 全球热带气旋最佳路径和预报数据
- **JMA（日本气象厅）**：https://www.jma.go.jp — 西太平洋台风最佳路径数据
- **中国气象局台风网**：http://typhoon.nmc.cn — 西太平洋和南海台风实时路径

#### 13. 降水雷达与卫星数据
- **GPM（全球降水测量）**：https://gpm.nasa.gov — 全球降水实时监测（NASA）
- **风云卫星数据**：https://satellite.nsmc.org.cn — 中国风云卫星遥感数据
- **CMORPH降水分析**：NOAA提供的全球高分辨率降水分析产品

#### 14. 洪涝与地质灾害
- **GloFAS（全球洪水预警系统）**：https://www.globalfloods.eu — ECMWF全球洪水预报
- **GFDRR（全球减灾与恢复基金）**：灾害风险数据

#### 15. 东盟区域专用平台
- **ASMC（东盟专项气象中心）**：https://www.aseanspecialisedmet.org — 东盟区域气候展望、季风监测
- **AHA Centre（东盟人道主义援助协调中心）**：灾害预警与协调

## 项目亮点（评审加分项）

1. **东盟特色鲜明**：多语言能力直接对标大赛"面向东盟"核心导向，是同赛道唯一真正的多语言产品
2. **技术架构先进**：采用LLM Agent架构（而非简单ChatBot），具备工具调用、自主规划、记忆管理能力
3. **双模交互创新**：对话问答（用户主动）+ AI定时推送（系统主动）双模式，实现"信息找人"而非"人找信息"
4. **移动端优先**：响应式H5设计，微信生态无缝接入
5. **落地路径清晰**：先做中文+英语+越南语三语言MVP，再扩展其他语言
6. **数据合规**：使用主办方提供的数据和公开API，无数据合规风险
7. **成果转化潜力**：可对接广西气象局"中国—东盟气象之家"平台
